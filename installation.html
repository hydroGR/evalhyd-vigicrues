Installation

Il est recommandé d’installer certaines dépendances au préalable :

Installer evalhyd avec conda
$ conda install -c conda-forge evalhyd-python
Installer libhydro avec pip
$ pip install git+https://gitlab.com/vigicrues/hydro3/libhydro

Il est ensuite possible d’installer evalhyd-vigicrues et ses autres dépendances via pip :

Installer evalhyd-vigicrues avec pip
$ pip install git+https://gitlab.irstea.fr/HYCAR-Hydro/evalhyd/evalhyd-vigicrues
